Geoportal Server

Version 2.5.1

Geoportal Server help user quickly find relevant, usable geospatial resources.

Geoportal Server is a standards-based, open source product that enables discovery and use of geospatial resources, including data and services. Please visit Geoportal Server wiki page for more information.
